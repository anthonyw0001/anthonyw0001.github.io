Task List

JavaScript

This is a program I made in May 2021, it's a to-do list. You can write tasks and enter them, filter particular tasks, delete one task at a time, or delete all of them together. The program also sends a browser alert, asking the user to confirm if they want to delete all tasks. Finally, if you refresh or close the page and come back, the program uses the browser storage to remember the tasks that were input.
