Creating a Basic Web Server on LocalHost
using LAMP and WordPress

Linux, Apache, MySQL, PHP, WordPress

This is a video I worked on with my fellow test trainee, Anthon Onn. We took a spare Thinkpad laptop and installed Linux Mint, Apache Server, MySQL, and PHP on it, creating a basic server which we installed WordPress onto. Despite some mistakes and troubleshooting, it took us less than an hour to get it up and running from scratch.
