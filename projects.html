Here is where I keep my JavaScript code, you can interact with the programs, and read the code underneath.

Task List

This is a program I made in May 2021, it's a to-do list. You can write tasks and enter them, filter particular tasks, delete one task at a time, or delete all of them together. The program also sends a browser alert, asking the user to confirm if they want to delete all tasks. Finally, if you refresh or close the page and come back, the program uses the browser storage to remember the tasks that were input.





Loan Calculator

This is another program I made, in June 2021. It takes the values for a loan and outputs a calculation for the amount to be paid back at the end of a term. It has an animated loading symbol to pad the user experience, to show that the computer is really thinking about the user's finances.
